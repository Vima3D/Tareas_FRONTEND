Ejercicio 2 - Clase 17 JAVASCRIPT

Auto

Propiedades

  • encendido (booleano), inicializa en false
  • velocidad (número), inicializa en 0
  • marca (string)
  • modelo (número)
  • patente (string)

Constructor

  • Pide como argumentos marca, modelo y patente y los asigna a sus propiedades

Métodos

  • arrancar(): pone encendido en true
  • apagar(): pone encendido en false
  • acelerar(): suma 10 a velocidad
  • desacelerar(): resta 10 a velocidad
  • toString(): devuelve ${marca} ${modelo}, patente ${patente}

Observaciones

  • Sólo se puede acelerar o desacelerar si el auto está encendido
  • Sólo se puede apagar si la velocidad es 0
  • La velocidad mínima es 0
